# Watch-TV-on-VLC
An AutoHotKey Script to watch TV on VLC Media Player

How to use: 
1. Download the files and place both "watch_tv.ahk" and "channel_list.txt" in the same directory.
2. Open "watch_tv.ahk" by double clicking on it.
3. Select the desired option from the dropdown menu and then press "Watch" to open the playlist in VLC Media Player.
4. Enjoy!
