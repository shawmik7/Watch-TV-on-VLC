# Watch-TV-on-VLC
An AutoHotKey Script to watch TV on VLC Media Player

**How to use: **
1. Download AutoHotKey v2 from https://www.autohotkey.com/
2. Download the files and place both "watch_tv.ahk" and "channel_list.txt" in the same directory.
3. Open "watch_tv.ahk" by double clicking on it.
4. Select the desired option from the dropdown menu and then press "Watch" to open the playlist in VLC Media Player.
5. Enjoy! 

**How to update the channel list: **
The format for the "channel_list.txt" file is: "text_shown_on_the_dropdown_list": "link_for_the_playlist" 
You can get the links from https://github.com/iptv-org/iptv/blob/master/PLAYLISTS.md
